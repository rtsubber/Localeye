Authentication

All requests require an API key passed via the X-API-Key header. Register for a free key below.

POST /v1/register
Create a free API key
# Get your free API key
curl -X POST https://ron-system-product-name.tail38a93d.ts.net/localeye/v1/register?email=you@example.com

# Response
{
  "key_id": "ley_dG9wX3NlY3JldF9rZXk",
  "email": "you@example.com",
  "tier": "free",
  "daily_limit": 10
}

Then include the key in all subsequent requests:

curl https://ron-system-product-name.tail38a93d.ts.net/localeye/v1/verify-web-presence \
  -H "X-API-Key: ley_your_key_here"

Verify Web Presence

The core endpoint. Fetches a URL from a residential IP on real hardware, renders JavaScript, and returns the content. Perfect for verifying business listings, checking hours, or extracting information that a datacenter IP would get blocked on.

POST /v1/verify-web-presence
Fetch and verify a URL with residential IP + GPU rendering

Request Body

FieldTypeDescription
urlstringURL to verify (required)
target_elementstringWhat to extract: text, html, or screenshot (default: text)
viewportstringBrowser viewport for screenshots (default: 1280x720)
wait_secondsfloatSeconds to wait for JS rendering (default: 2.0)
# Verify a business website exists and get its content
curl -X POST https://ron-system-product-name.tail38a93d.ts.net/localeye/v1/verify-web-presence \
  -H "X-API-Key: ley_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://joespizza.com",
    "target_element": "text"
  }'

# Response
{
  "status": "verified",
  "url": "https://joespizza.com",
  "content": "Joe's Pizza - Authentic NY Style Pizza Since 1985...",
  "response_time_ms": 1240,
  "verified_at": "2026-05-16T22:00:00Z"
}

Visual Verify

Same as web presence verification, but returns a screenshot of the page. Your agent can see what a human sees — layouts, images, "OPEN" signs, business hours displays.

POST /v1/visual-verify
Verify a URL and return a screenshot
# Take a screenshot of a business page
curl -X POST https://ron-system-product-name.tail38a93d.ts.net/localeye/v1/visual-verify \
  -H "X-API-Key: ley_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://joespizza.com",
    "viewport": "1280x720"
  }'

# Response
{
  "status": "verified",
  "url": "https://joespizza.com",
  "screenshot_url": "https://...localeye/screenshots/abc123.png",
  "response_time_ms": 3200
}

Phone Verify

Local-Eye can call a business phone number and ask a question. Use it to verify hours, confirm services, or check if a business is real. The AI call transcribes the response in real-time.

POST /v1/phone-verify
Call a business and ask a question

Request Body

FieldTypeDescription
business_phonestringPhone number to call (required)
questionstringQuestion to ask (default: "Are you open right now?")
business_namestringName of the business to ask for (optional)
# Call a business to verify hours
curl -X POST https://ron-system-product-name.tail38a93d.ts.net/localeye/v1/phone-verify \
  -H "X-API-Key: ley_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "business_phone": "+1-555-123-4567",
    "question": "Are you open right now and what are your hours today?",
    "business_name": "Joe's Pizza"
  }'

# Response
{
  "status": "completed",
  "call_sid": "CAabc123...",
  "transcript": "Yes, we're open today until 10 PM...",
  "duration_seconds": 23,
  "verified": true
}
